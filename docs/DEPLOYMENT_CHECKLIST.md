# LLM智能路由层部署检查清单

使用此清单确保部署过程完整无误。

## 部署前检查

### 硬件资源
- [ ] Router Gateway服务器：3-5台，4核8GB+
- [ ] 轻量模型服务器：10-20台，16GB显存/32GB内存
- [ ] 分类器服务器（可选）：4-8台，16GB显存
- [ ] 网络：内网延迟<5ms，带宽≥1Gbps

### 软件环境
- [ ] 操作系统：Ubuntu 20.04+/CentOS 7+
- [ ] Python：3.10+
- [ ] CUDA：11.8+（如使用GPU）
- [ ] Docker：20.10+（可选）
- [ ] Nginx：1.18+

### 准备工作
- [ ] 创建部署用户 `llm-router`
- [ ] 配置SSH免密登录
- [ ] 同步服务器时间
- [ ] 配置防火墙规则

---

## 部署步骤

### 第一步：Router Gateway（每台）
- [ ] 克隆代码仓库
- [ ] 创建Python虚拟环境
- [ ] 安装依赖包
- [ ] 配置`.env`文件
- [ ] 配置`config/rules.yaml`
- [ ] 创建日志目录
- [ ] 测试启动（开发模式）
- [ ] 配置Systemd服务
- [ ] 启动并验证服务
- [ ] 测试API接口

### 第二步：轻量模型池（每台）
- [ ] 安装vLLM
- [ ] 下载Qwen2.5-7B-Instruct模型
- [ ] 创建启动脚本
- [ ] 配置Systemd服务
- [ ] 启动vLLM服务
- [ ] 验证模型服务
- [ ] 记录服务地址和端口

### 第三步：分类器服务（如单独部署）
- [ ] 安装vLLM
- [ ] 启动分类器服务（使用不同端口）
- [ ] 更新Router配置
- [ ] 验证分类器连通性

### 第四步：Nginx负载均衡
- [ ] 安装Nginx
- [ ] 配置upstream（所有Router实例）
- [ ] 配置限流和超时
- [ ] 配置SSL证书（可选）
- [ ] 测试Nginx配置
- [ ] 重载Nginx

### 第五步：Redis缓存（可选）
- [ ] 安装Redis
- [ ] 配置密码和持久化
- [ ] 启动Redis服务
- [ ] 测试Redis连接
